## Notes on Citation Practice

**In-text citations should follow this format:**
- Single author: (Quine 1960)
- Two authors: (Ladyman & Ross 2007)
- Three+ authors: (French et al. 2003)
- Multiple works: (Quine 1960, 1980a; Putnam 1971)
- Page numbers when quoting: (Ladyman 1998: 412)

**Bibliography formatting:**
- Alphabetical by author surname
- Chronological within same author
- Journal articles: Author, Year. "Title." *Journal*, Volume(Issue): Pages.
- Books: Author, Year. *Title*. Place: Publisher.
- Edited volumes: Author, Year. "Title." In Editor (ed.), *Book Title*. Place: Publisher, pp. Pages.

**Abbreviations:**
- ed./eds. = editor/editors
- trans. = translator
- 2nd ed. = second edition
- pp. = pages
- vol. = volume

**Online sources:**
- Include URL for stable archives (Stanford Encyclopedia, PhilSci Archive)
- Include DOI when available
- Note retrieval date only if content may change
