🔬 GIFT Translator

Bidirectional translation between Standard Model and Geometric Information Field Theory

🚀 Try it locally

Clone and run the translator on your machine:

git clone https://github.com/gift-framework/gift.git cd gift pip install flask numpy python run_web_interface.py

Then open http://localhost:5000

📝 Translation Examples

Einstein's equation:
Standard Model: E = mc²
GIFT: E = ξ·τ·c²·m
Fine structure constant:
Standard Model: α = e²/(4πε₀ℏc)
GIFT: α⁻¹ = ζ₃ × 114 - 1/24
Weinberg angle:
Standard Model: sin²θ_W = g'²/(g² + g'²)
GIFT: sin²θ_W = ζ₂ - √2

💻 Command Line Usage

# Interactive mode python -m gift_translator.cli --interactive # Direct translation python -m gift_translator.cli "E = mc²" --from SM --to GIFT
