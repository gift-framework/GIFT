Formats de citation pour le cadre K₇ v3.5 (anciennement GIFT).

## Citation logicielle (recommandée)

### BibTeX

```bibtex
@software{k7_framework_v35,
  title   = {The K₇ Framework (formerly GIFT)},
  author  = {de La Fournière, Brieuc},
  year    = {2026},
  url     = {https://github.com/Arithmon/K7},
  doi     = {10.5281/zenodo.16891489},
  version = {3.5},
  license = {MIT},
  note    = {95 observables, 0.99\% mean deviation on 33 Type I relations (PDG 2024 / NuFIT 6.1), 140 conjuncts certified in Lean 4, 15 axioms (4 main-chain + 11 interval-arithmetic), Donaldson certificate pipeline scoped by docs/analytic_status.md}
}
```

### Style APA

```
de La Fournière, B. (2026). The K₇ Framework (formerly GIFT) (Version 3.5) [Logiciel]. Zenodo. https://doi.org/10.5281/zenodo.16891489
```

### Style Chicago

```
de La Fournière, Brieuc. "The K₇ Framework (formerly GIFT)." Version 3.5. Zenodo, 2026. https://doi.org/10.5281/zenodo.16891489.
```

## Utilisation en publication

Lorsque vous utilisez des prédictions K₇ :

1. **Citer le cadre** : utilisez la citation logicielle ci-dessus
2. **Citer des résultats spécifiques** : utilisez l'entrée du document ou de l'article compagnon correspondant
3. **Préciser la version** : incluez toujours le numéro de version (v3.5)
4. **Lier le dépôt** : incluez l'URL GitHub pour la reproductibilité

### Exemple

> « Nous comparons nos mesures avec les prédictions du cadre K₇ (anciennement GIFT) [1], qui dérive les paramètres du Modèle Standard de la topologie E₈×E₈. La prédiction K₇ pour la phase de violation CP est δ_CP = 197° [2]. »
>
> [1] de La Fournière, B. "The K₇ Framework v3.5," 2026, https://github.com/Arithmon/K7
>
> [2] de La Fournière, B. "K₇ Framework S2: Complete Derivations," 2026.
