# Pour commencer

## Quel est votre profil ?

Choisissez la voie qui vous correspond le mieux :

### Je suis curieux de science (sans maths nécessaires)
Commencez par **[GIFT pour Tout le Monde](GIFT-for-Everyone.fr.html)** : un guide complet utilisant des analogies du quotidien. Puis consultez la **[FAQ](FAQ.fr.html)** pour les questions courantes.

### Je suis physicien
Lisez **[Pour les physiciens](For-Physicists.html)** pour une vue d'ensemble technique : architecture E₈×E₈, compactification sur K₇, et les 33 prédictions. Plongez ensuite dans l'**[article principal](Paper-Main-Framework.html)**.

### Je suis géomètre / mathématicien
Lisez **[Pour les géomètres](For-Geometers.html)** pour le pipeline computationnel : construction TCS, métriques PINN, extraction spectrale, et ponts vers Lean 4. L'article **[Métrique G₂ explicite](Paper-Explicit-G2-Metric.html)** est votre point d'entrée.

### Je m'intéresse à la vérification formelle
Lisez **[Pour les experts en formalisation](For-Formalization-Experts.html)** pour l'architecture Lean 4 : 130 fichiers, 7 axiomes, certificat à 213 conjonctions, zéro sorry. Voir **[Formalisation Lean](Lean-Formalization.html)** pour la vue d'ensemble.

### Je veux vérifier les chiffres
Allez directement à la **[Référence des observables](Observable-Reference.html)** (89 observables) et aux **[Preuves statistiques](Statistical-Evidence.html)** (validation à 7 composantes, recherche exhaustive sur 3M+ configurations).

### Je veux citer GIFT
Voir le **[Guide de citation](Citation-Guide.html)** pour les formats BibTeX, APA et DOI.

---

## Ordre de lecture recommandé

1. **[GIFT pour Tout le Monde](GIFT-for-Everyone.fr.html)** ou **[Pour les physiciens](For-Physicists.html)** (selon votre profil)
2. **[Article principal](Paper-Main-Framework.html)** : la théorie complète
3. **[Référence des observables](Observable-Reference.html)** : toutes les prédictions dans un tableau
4. **[Preuves statistiques](Statistical-Evidence.html)** : pourquoi les chiffres sont importants
5. **[Formalisation Lean](Lean-Formalization.html)** : preuves vérifiées par machine
