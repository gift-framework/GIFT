## 1. Dark Matter Sector Predictions

### 1.1 Particle Masses

The framework predicts two-component dark matter with specific masses:

**Lightest component**:
```
m_χ₁ = √M₁₃ = √8191 = 90.5 GeV
```

where M₁₃ = 2¹³ - 1 = 8191 (13th Mersenne prime), and exponent 13 = Weyl_factor + rank(E₈) = 5 + 8.

**Heavier component**:
```
m_χ₂ = τ × √M₁₃ = 3.897 × 90.5 = 352.7 GeV
```

where τ = 10416/2673 is the hierarchical scaling parameter derived from manifold geometry.

**Mass ratio**: m_χ₂/m_χ₁ = τ = 3.897 (fixed by topology)

### 1.2 Relic Abundance Consistency

Framework prediction for total dark matter density:
```
Ω_DM = (π + γ)/M₅ = 3.71881/31 = 0.11996
```

Experimental value: Ω_DM h² = 0.1199 ± 0.0027 (Planck 2018)
Deviation: 0.032%

Two-component model must satisfy: Ω_χ₁ + Ω_χ₂ = 0.120

### 1.3 Direct Detection Prospects

**Target mass**: m_χ₁ = 90.5 GeV

**Current limits** (XENON1T):
- Spin-independent cross-section: σ_SI < 10⁻⁴⁶ cm² (90 GeV mass)

**Next-generation experiments**:
- XENONnT (2024-2028): σ_SI sensitivity ~10⁻⁴⁸ cm²
- LZ (2024-2028): σ_SI sensitivity ~10⁻⁴⁸ cm²
- DARWIN (2030+): σ_SI sensitivity ~10⁻⁴⁹ cm²

**Prediction**: If Higgs portal coupling λ_portal ~ 0.01-0.1, detection possible within next 5-10 years.

### 1.4 Collider Searches

**Target masses**: m_χ₁ = 90.5 GeV, m_χ₂ = 352.7 GeV

**LHC constraints**:
- Monojet + MET searches: Sensitive to ~100-400 GeV range
- Current limits: No signal (consistent with predictions)

**HL-LHC prospects** (2027-2035):
- Integrated luminosity: 3000-4000 fb⁻¹
- Mass reach: ~500 GeV for χ₂
- Sensitivity to two-component signature if production cross-section σ > 0.1-1 fb

**Kinematic constraint**: H → χ₁χ₁ forbidden (2×m_χ₁ = 181 GeV > m_H = 125 GeV)

**Alternative channels**: Z-portal, photon mixing, direct pair production

### 1.5 Indirect Detection

**Annihilation signatures**:
- χ₁χ₁ → γγ: Monochromatic line at E_γ = 90.5 GeV
- χ₂χ₂ → γγ: Monochromatic line at E_γ = 352.7 GeV
- Mixed: χ₁χ₂ → products

**Current experiments**:
- Fermi-LAT: Sensitive to ~100 GeV γ-ray lines
- H.E.S.S., VERITAS: Ground-based Cherenkov telescopes
- HAWC: Water Cherenkov detector

**Future prospects**:
- CTA (Cherenkov Telescope Array, 2025+): Improved sensitivity 100 GeV - 1 TeV
- Optimal for m_χ₁ and m_χ₂ searches

**Interpretation**: Detection of two distinct γ-ray lines at predicted energies would strongly support framework.

## 3. Neutrino Sector Tests

**Framework prediction**: δ_CP = (3π/2) × (4/5) = 216°

**Current status**:
- Experimental: δ_CP = 197° ± 24° (NOvA 2024)
- Theory within 1σ
- Large experimental uncertainty (~12%)

**Future measurements**:
- T2K (2024-2026): δδ_CP ~ ±15°
- NOvA (2024-2026): δδ_CP ~ ±20°
- DUNE (2029+): δδ_CP ~ ±5° (precision era)
- Hyper-Kamiokande (2027+): δδ_CP ~ ±7°

**Testability**: Framework prediction will be tested at 2-3% level by 2030.

### 3.2 Neutrino Mass Ordering

**Framework implication**: Normal ordering preferred (based on θ₂₃ = 85/99 structure)

**Current status**:
- Global fits favor normal ordering (~3σ)
- Not definitive

**Upcoming experiments**:
- JUNO (2024+): 3-4σ determination within 6 years
- Hyper-Kamiokande (2027+): Independent confirmation

**Framework consistency**: Check whether mass ordering determination aligns with predicted angle structure.

### 3.3 Absolute Mass Scale

**Framework limitation**: Predicts mixing angles but not absolute mass scale (requires see-saw mechanism connection).

**Future**: β-decay experiments (KATRIN, Project 8) and cosmological observations (CMB, LSS) will constrain Σm_ν. Framework should predict mass ratios given absolute scale.

## 4. Cosmological Observables

### 4.1 Hubble Constant

**Framework prediction**:
```
H₀ = H₀^CMB × (ζ(3)/ξ)^β₀ = 67.36 × 1.0827 = 72.93 km/s/Mpc
```

**Current measurements**:
- CMB (Planck): H₀ = 67.36 ± 0.54 km/s/Mpc (early universe)
- Local (SH0ES): H₀ = 73.04 ± 1.04 km/s/Mpc (late universe)
- Tension: 4.4σ discrepancy

**Framework explanation**: Geometric correction factor (ζ(3)/ξ)^β₀ = 1.0827 from K₇ compactification.

**Prediction vs. local**: 72.93 vs. 73.04 km/s/Mpc (deviation 0.145%)

**Future observations**:
- JWST: High-redshift distance ladder
- Euclid (2024-2030): Independent H₀ measurement from weak lensing
- Vera Rubin Observatory (2025-2035): Large-scale structure constraints
- Expected resolution: ΔH₀ ~ ±0.5 km/s/Mpc by 2030

**Testability**: If local H₀ measurement converges to ~73 km/s/Mpc with reduced uncertainty, framework prediction validated. If converges to ~67 km/s/Mpc, tension remains unexplained.

### 4.2 Dark Energy Density

**Framework prediction**: Ω_DE = ln(2) × 98/99 = 0.6861

- Expected precision: 0.3% by 2030

**Testability**: Framework prediction within current uncertainties. Will be tested at 0.2-0.3% level.

### 4.3 Scalar Spectral Index

## 9. Falsification Criteria

### 9.1 Strong Falsification

Framework would be **strongly challenged** if:

1. **Dark matter masses** detected but m_χ ≠ 90.5 GeV or 352.7 GeV (±10%)
2. **Quark mass ratios** converge to values inconsistent with predictions at 0.5% level
3. **δ_CP** measured precisely to ≠ 216° (±10° at 5σ)
4. **H₀ tension** remains after precision improvements contradict geometric correction

### 9.2 Moderate Falsification

Framework would be **moderately challenged** if:

1. **α⁻¹ offset** cannot be explained by geometric corrections from running
2. **Neutrino mass ordering** found to be inverted (contrary to θ₂₃ structure implications)
3. **Multiple observables** consistently deviate by >1% as precision improves

### 9.3 Validation Criteria

Framework would be **strongly supported** if:

1. **Dark matter detected** at m_χ₁ = 90.5 ± 5 GeV (revolutionary confirmation)
2. **Second component** at m_χ₂ = 353 ± 20 GeV with mass ratio τ ≈ 3.9
3. **All observable predictions** remain within 1% as experimental precision improves
4. **Pattern structures** (Mersenne-Fermat, odd zeta, 17 universality) hold across new measurements
