\section{Statistical Methods}

\subsection{Chi-Square Analysis}

For $N$ observables with predictions $\{P_i\}$ and measurements $\{M_i \pm \sigma_i\}$:
\begin{equation}
\chi^2 = \sum_i \frac{(P_i - M_i)^2}{\sigma_i^2}
\end{equation}

\subsection{Pull Distribution}

Pull for observable $i$:
\begin{equation}
z_i = \frac{P_i - M_i}{\sigma_i}
\end{equation}

Expected for correct theory: $z \sim \mathcal{N}(0,1)$.

\section{Falsification Philosophy}

\subsection{Scientific Standards}

A viable physical theory must be falsifiable. GIFT adheres to this principle by providing:

\begin{enumerate}
    \item \textbf{Exact predictions} that allow no deviation
    \item \textbf{Quantitative bounds} for all other predictions
    \item \textbf{Clear experimental signatures} for testing
    \item \textbf{Explicit exclusions} of alternative scenarios
\end{enumerate}

\subsection{Classification of Tests}

\begin{description}
    \item[Type A (Absolute):] Violation of topological identity falsifies framework immediately
    \begin{itemize}
        \item $\Ngen = 3$ (generation number)
        \item Exact rational relations ($\sinW = 3/13$, $\tau = 3472/891$)
        \item Exact integer relations
    \end{itemize}

    \item[Type B (Bounded):] Deviation beyond stated tolerance is problematic
    \begin{itemize}
        \item Most observables with finite precision
        \item Statistical significance required (typically $> 5\sigma$)
    \end{itemize}

    \item[Type C (Directional):] Qualitative predictions
    \begin{itemize}
        \item Existence/non-existence of particles
        \item Sign of CP violation
    \end{itemize}
\end{description}
