# Contributing

Thank you for your interest in contributing to GIFT.

## Documentation Contributions

This repository contains the theoretical documentation for GIFT v3.3. Contributions welcome:

- **Clarifications**: Improve explanations of concepts
- **Corrections**: Fix errors in derivations or references
- **Translations**: Other formats or languages

## How to Contribute

1. Open an issue describing your proposed change
2. Fork the repository and make changes
3. Submit a pull request with a clear description

## Scientific Standards

- Maintain mathematical correctness
- Use appropriate status classifications (PROVEN, TOPOLOGICAL, DERIVED, etc.)
- Report both strengths and limitations
- Include references for external results
